How does this work?

3 simple steps to show you how to add Typicons icon font-family to your website!

1. Add the icon font(s) to your collection:


2. Import the icon fonts(s) into your css and add the font-family('s):

@import url(http://weloveiconfonts.com/api/?family=typicons|brandico);

/* brandico */
[class^="brandico-"]:before, 
[class*="brandico-"]:before {
  font-family: 'brandico', sans-serif;
}

/* typicons */
[class^="typicons-"]:before, 
[class*="typicons-"]:before {
  font-family: 'Typicons', sans-serif;
}

3. Burn baby, burn!

<!-- Single Element -->
<span class="typicons-puzzle"></span>
        
<!-- Group of Elements -->
<ul>
  <li class="brandico-twitter-bird"></li>
  <li class="brandico-odnoklassniki"></li>
  <li class="brandico-icq"></li>
  <li class="brandico-github"></li>
</ul>
