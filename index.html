How does this work?

3 simple steps to get open source icon fonts onto your website!

1. Add Icon Fonts to your collection:


2. Import them into your css and add the font-family:

@import url(http://weloveiconfonts.com/api/?family=typicons|brandico);

.typicons {
  font-family:'Typicons', sans-serif;
}

.brandico {
  font-family:'brandico', sans-serif;
}

3. Burn baby, burn!

<!-- Single Element -->
<span class="brandico brandico-twitter-bird">
        
<!-- Group of Elements -->
<ul class="brandico">
  <li class="brandico-twitter-bird"></li>
  <li class="brandico-odnoklassniki"></li>
  <li class="brandico-icq"></li>
  <li class="brandico-github"></li>
</ul>
